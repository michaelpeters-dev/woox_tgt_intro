# Woo X Trading at GT Intro Project

Ingests level-2 ETH-USDT perpetual futures data via the Woo X websocket and prints the state of the order book after every market data update.

## Requirements

- Rust
- Cargo

You can verify your installation with:
```bash
rustc --version
cargo --version
```

Building the Project:
```bash
cargo build
```

Running the Project:
```bash
cargo run
```
